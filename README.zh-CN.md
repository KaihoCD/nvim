# Neovim 配置

[English](./README.md) | 中文文档

本仓库包含针对不同 Neovim 版本的配置。请根据你的 Neovim 版本选择对应的分支。

## 可用版本

### Beta-1 分支

**Neovim 版本**: >= 0.10（推荐：0.11.x）

**状态**: 稳定但可能存在兼容性问题

> **注意**: 此配置可能包含过时的设置。如遇到错误，请自行解决。

**快速访问**:

- [查看 Beta-1 分支](https://github.com/KaihoCD/nvim/tree/beta-1)

---

### Beta-2 分支（推荐）

**Neovim 版本**: >= 0.12

**状态**: 长期维护中

> **积极维护**: 此版本正在积极维护和定期更新。

**快速访问**:

- [查看 Beta-2 分支](https://github.com/KaihoCD/nvim/tree/beta-2)

---
