# Neovim Configuration

This repository contains multiple Neovim configurations for different versions. Please choose the branch that matches your Neovim version.

### Beta-1 Branch

**Status**: Stable but may have compatibility issues

> **Notice**: This configuration may contain outdated settings. If you encounter errors, please resolve them on your own.

**Quick Access**:

- [View Beta-1 Branch](https://github.com/KaihoCD/nvim/tree/beta-1)

### Beta-2 Branch (Recommended)

**Neovim Version**: >= 0.12

**Status**: Long-term maintenance

> **Actively Maintained**: This version is actively maintained and updated regularly.

**Quick Access**:

- [View Beta-2 Branch](https://github.com/KaihoCD/nvim/tree/beta-2)
